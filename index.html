Web Developer & AI Automation Specialist

Soy Daniel, desarrollador web con especialización en automatización inteligente con inteligencia artificial. Diseño e implemento workflows automatizados usando herramientas como N8N, aplicando prompt engineering y protocolos como MCP (Model Context Protocol) para integrar modelos de IA en procesos reales, ágiles y escalables. Con base en el ecosistema JavaScript (React.js, Node.js, MongoDB), combino desarrollo tradicional con tecnologías low-code/no-code para construir soluciones modernas que aporten valor real y productividad. Me mantengo en constante evolución, buscando aplicar inteligencia artificial no solo como asistente, sino como una herramienta clave para transformar procesos digitales y potenciar negocios.
